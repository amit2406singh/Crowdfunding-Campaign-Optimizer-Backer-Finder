# CrowdAI – Crowdfunding Campaign Optimizer

An intelligent, full-stack platform that helps campaign creators **plan, launch, and optimize** their crowdfunding campaigns using AI and data science.

## ✨ Features

- 🔭 **Explore Gallery** – Browse all active campaigns with live progress, pledge support, and quick-fund capabilities
- 📊 **Dashboard & Metrics** – Real-time funding velocity, backer counts, and campaign health indicators
- 🤖 **AI Campaign Planner** – Generate optimized campaign strategies powered by Groq LLM
- ✍️ **Page Creator & Media** – AI-assisted copywriting for titles, descriptions, and reward tiers
- 📣 **Launch Marketing Hub** – Pre-launch checklists, social media kits, and campaign readiness scoring
- 💬 **Backer Engagement** – AI-generated backer announcements and community update templates
- 🧪 **A/B Test Sandbox** – Test campaign copy variants and analyze which performs better
- 📈 **ML Prediction Engine** – Velocity prediction, risk assessment, success probability scoring, and scenario trajectories (pessimistic / expected / optimistic)

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19 + TypeScript + Vite |
| Backend (API & AI) | Node.js + Express + Groq SDK |
| Backend (ML & Analytics) | Python + FastAPI + scikit-learn + NumPy + Pandas |
| Database | MongoDB (via Mongoose) |
| LLM | Groq (LLaMA-based) |
